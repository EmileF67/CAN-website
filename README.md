# CAN Website | HTML CSS JavaScript

Petit site de démonstration pour la gestion d'une application CAN : affichage de cartes d'embarquement, gestion des factures, visualisation de statistiques et tableau de bord.

Pages (dossier `html`):
- `accueil.html` : page d'accueil du site.
- `cartes_embarquement.html` : création et affichage des cartes d'embarquement.
- `facture.html` : affichage/génération des factures.
- `statistiques.html` : visualisation des statistiques (graphiques).
- `tableau_bord.html` : interface principale / tableau de bord.

Scripts (dossier `js`):
- `jscarte.js` : logique et interactions pour les cartes d'embarquement.
- `jsfact.js` : fonctions pour la génération et le calcul des factures.
- `jsstat.js` : collecte et rendu des données statistiques.
- `jstabbord.js` : comportements et contrôles du tableau de bord.

Styles:
- `css/style1.css` : styles globaux (mise en page, couleurs, responsive).

Dossiers de ressources:
- `img/` : images et icônes utilisées par les pages.
- `video/` : vidéos d'illustration.
- `download/` : fichiers générés à télécharger (ex. PDF).

Utilisation rapide : ouvrir les fichiers `html/*.html` dans un navigateur. Modifier les fichiers `js/` et `css/style1.css` pour adapter le comportement et l'apparence.
